## Usage
> plugin config
```javascript
const options = {
  // required
  accessKeyId: 'your ali accessKeyId',
  accessKeySecret: 'your ali accessKeySecret',
  region: 'your ali region',
  bucket: 'your ali bucket',
  // not required
  https: true,
  domain: 'your domain', // eg: domain.com
  name: 'your filename' // eg: filename/
}
```
